# House Price Prediction

## Overview
Predicts house prices using Machine Learning.

## Features
- Enter house details
- Predict house price instantly
- Streamlit interface

## Technologies
- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- NumPy

## Dataset
https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

## Model trained on
XGBoost Regressor
Linear Regression

## model uses Linear Regression

## How to Run

pip install -r requirements.txt

streamlit run app.py
